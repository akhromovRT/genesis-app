# Knowledge Base — Genesis

База знаний для обучения Genesis Coach. Контент структурирован для использования AI-ботом при генерации персональных рекомендаций.

## Структура

```
knowledge_base/
├── README.md              — этот файл
└── books/                 — извлечённые протоколы из книг
    ├── biohacking-po-zhenski-overview.md       — обзор книги, структура, ключевые тезисы
    ├── biohacking-po-zhenski-protocols.md       — все протоколы: ритмы, сон, питание, стресс, БАД, спорт
    ├── biohacking-po-zhenski-12-principles.md   — 12 принципов биохакинга с привязкой к Genesis
    ├── biohacking-po-zhenski-bioage-tests.md    — 15 тестов биологического возраста + формулы КоэфСС
    └── biohacking-po-zhenski-longevity-pincode.md — ПИН-код долголетия, системный подход
```

## Как использовать для бота

1. **overview** — для общего контекста и философии ответов
2. **protocols** — основной файл для генерации рекомендаций (циркадные ритмы, питание, сон, стресс, БАД, спорт)
3. **12-principles** — для проверки системности рекомендаций
4. **bioage-tests** — для модуля самодиагностики биологического возраста
5. **longevity-pincode** — для мотивационного и образовательного контента
