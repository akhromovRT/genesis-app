# 15 тестов на определение биологического возраста

> Источник: Г. Хусаинова «Биохакинг по-женски» — Приложение 2

## Методика расчёта

Провести как можно больше тестов → сложить полученные «возрасты» → разделить на количество тестов = примерный биологический возраст.

**Правила:**
- Тесты НЕ делать подряд — восстанавливаться между ними
- Распределить на несколько часов/дней
- Не переходить рубеж комфорта
- Желательно делать в присутствии другого человека

---

## Тесты

### 1. Тест на пульс
Быстро подняться с 1-го на 4-й этаж (~80 шагов/мин). Измерить пульс сразу и через 2 мин.

| Пульс сразу | Пульс через 2 мин | Возраст |
|-------------|-------------------|---------|
| до 100 | до 80 | 20–30 |
| 100–120 | 80–100 | 30–40 |
| 120–140 | 100–120 | 40–50 |
| >140 | >120 | 50+ |

### 2. Тест на частоту наклонов (выносливость)
Стоя прямо, доставать пол кончиками пальцев (колени не сгибать). Максимум за 1 минуту.

| Количество | Возраст |
|-----------|---------|
| >50 | 20 |
| 40–50 | 30 |
| 30–40 | 40 |
| 20–30 | 50 |
| <20 | 60+ |

### 3. Тест на наклон (гибкость)
С прямыми ногами попытаться коснуться пола.

| Результат | Возраст |
|----------|---------|
| Ладони на полу | 20–30 |
| Пальцы касаются пола | 30–40 |
| Кончики пальцев до пола | 40–50 |
| До голеней | 50–60 |
| До коленей | 60+ |

### 4. Тест на вестибулярный аппарат
Пятку одной ноги к колену другой, глаза закрыты. Засечь время.

| Время | Возраст |
|-------|---------|
| >30 сек | 20 |
| 20–30 сек | 30–40 |
| 10–20 сек | 40–50 |
| 5–10 сек | 50–60 |
| <5 сек | 60+ |

### 5. Тест на частоту дыхания
Количество вдохов/выдохов за 1 минуту в обычном состоянии.

| Вдохов/мин | Возраст |
|-----------|---------|
| 14–15 | 20–30 |
| 16–17 | 30–40 |
| 18–20 | 40–50 |
| 21–25 | 50–60 |
| >25 | 60+ |

### 6. Тест на задержку дыхания
**Проба Штанге:** вдохнуть полной грудью → задержать дыхание (не выдыхая).
**Проба Генче:** после восстановления (15 мин) — полностью выдохнуть → задержать.

| Задержка на вдохе | Задержка на выдохе | Возраст |
|-------------------|-------------------|---------|
| >50 сек | >30 сек | 20–30 |
| 40–50 сек | 20–30 сек | 30–40 |
| 30–40 сек | 15–20 сек | 40–50 |
| 20–30 сек | 10–15 сек | 50–60 |
| <20 сек | <10 сек | 60+ |

### 7. Тест со свечой
Зажечь свечу на расстоянии 1 м. Попробовать задуть с первого раза. Если не получилось — приблизиться.

| Расстояние | Возраст |
|-----------|---------|
| 1 м | 20–30 |
| 80 см | 30–40 |
| 60 см | 40–50 |
| 40 см | 50–60 |
| <40 см | 60+ |

### 8. Тест на сжатие кожи (щипок)
Крепко сжать кожу на тыльной стороне кисти на 5 сек (побелеет). Засечь время возврата цвета.

| Время | Возраст |
|-------|---------|
| <2 сек | 20–30 |
| 2–5 сек | 30–40 |
| 5–10 сек | 40–50 |
| 10–15 сек | 50–60 |
| >15 сек | 60+ |

### 9. Тест на сведение рук в замок
Одну руку сверху (от шеи), другую снизу (от лопатки) — сцепить в замок за спиной.

| Результат | Возраст |
|----------|---------|
| Замок легко | 20–30 |
| Пальцы касаются | 30–40 |
| Почти касаются | 40–50 |
| Далеко друг от друга | 50–60 |
| Невозможно | 60+ |

### 10. Тест с линейкой (скорость реакции)
Помощник держит линейку 50 см вертикально, отпускает внезапно. Поймать сжатием большого и указательного пальцев. На каком сантиметре поймали?

| Сантиметр | Возраст |
|----------|---------|
| <20 см | 20 |
| 20–25 см | 30 |
| 25–30 см | 40 |
| 30–35 см | 50 |
| >35 см | 60+ |

### 11. Тест на подъём туловища
Лёжа на спине, одновременно отрывать голову, плечи и спину от пола (руки вперёд, не упираясь). Количество за 1 минуту.

| Количество | Возраст |
|-----------|---------|
| >40 | 20 |
| 30–40 | 30 |
| 20–30 | 40 |
| 10–20 | 50 |
| <10 | 60+ |

### 12. Тест на глюкозу
Измерить глюкометром утром натощак (не пить, не двигаться перед тестом).

| Глюкоза (ммоль/л) | Возраст |
|-------------------|---------|
| 3.5–5.0 | 20–30 |
| 5.0–5.5 | 30–40 |
| 5.5–6.0 | 40–50 |
| 6.0–6.5 | 50–60 |
| >6.5 | 60+ |

### 13. Тест на давление крови
В спокойном состоянии, за час без физической активности.

| Давление | Возраст |
|---------|---------|
| 110/70 | 20–30 |
| 120/80 | 30–40 |
| 130/85 | 40–50 |
| 140/90 | 50–60 |
| >140/90 | 60+ |

### 14. Тест на приседания (выносливость)
Адекватные усилия, без боли и задыхания.

| Количество | Возраст |
|-----------|---------|
| >100 | 20 |
| 75–100 | 30 |
| 50–75 | 40 |
| 25–50 | 50 |
| <25 | 60+ |

### 15. Тест на индекс грации
Измерить окружность голени (ОГ) и окружность талии (ОТ).
**Формула:** (ОГ / ОТ) × 100

| Индекс | Возраст |
|--------|---------|
| >52 | 20–30 |
| 46–52 | 30–40 |
| 40–46 | 40–50 |
| 35–40 | 50–60 |
| <35 | 60+ |

## Формула коэффициента скорости старения (КоэфСС)

**По методу Горелкина и Пинхасова (для женщин):**

```
КоэфССж = (ОТ × Вес) / (ОБ × Рост² × (14.7 + 0.26 × (Возраст − 18) + 0.0012 × (Возраст − 18)²))
```

**Параметры:** Рост (м), Вес (кг), ОТ — обхват талии (см), ОБ — обхват бёдер (см)

**Результаты:**
- **0.95–1.05** — обычная скорость старения
- **< 0.94** — замедленное старение
- **> 1.06** — ускоренное старение

**Биологический возраст:**
```
БВж = (Возраст − 18) × КоэфССж + 18
```

## Что делать, если тело старше паспорта?

1. Увеличить двигательную активность (прогулки, плавание, скандинавская ходьба, йога)
2. Снизить уровень стресса
3. Начать приём препаратов для обмена веществ:
   - **Глутатион/Элтацин** (цистин + глицин + глутаминовая кислота) — регенерация
   - **Треонин + витамин В6** — когнитивные функции, скорость реакции
