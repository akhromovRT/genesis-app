## Позиционирование

**Genesis** — персональная операционная система здоровья и долголетия. Приложение проводит человека через три уровня самопознания:
**кто ты генетически → каков ты сейчас → кем ты становишься** — и на каждом уровне даёт конкретные протоколы изменений.

**Ключевое отличие от конкурентов:** не просто измерить и показать цифры, а рассказать *историю твоего здоровья* — причинно-следственную, понятную, с персональным планом действий.

## Архитектура: 3 фазы продукта

### Фаза 1 — Genetic Foundation (твой генетический паспорт)

**Что делает:**
- Пользователь загружает сырые данные из 23andMe / AncestryDNA / MyHeritage или заказывает партнёрский тест
- AI-движок анализирует ключевые SNP-кластеры по группам:
  - Метаболизм (MTHFR, APOE, FTO, TCF7L2)
  - Детоксикация (COMT, GSTP1, CYP450)
  - Воспаление / иммунитет (IL-6, TNF-α, CRP-полиморфизмы)
  - Долголетие (FOXO3, SIRT1, APOC3)
  - Спортивный ответ (ACTN3, ACE)
  - Сон и циркадные ритмы (PER3, CLOCK)
  - Микронутриенты (VDR, BCMO1 — витамин D, бета-каротин)

**Вывод для пользователя:** не «у тебя такой-то генотип», а **«Вот твой генетический профиль риска и потенциала»** — понятная инфографика + нарратив на языке жизненных ситуаций.

**Алгоритм:** RAG поверх базы GWAS-исследований + клинических рекомендаций (ClinVar, dbSNP), приоритизация по effect size и клинической значимости.

---

### Фаза 2 — Dynamic Biomarker Layer (живой слепок здоровья)

**Блок крови:**
- Импорт анализов (OCR фото/PDF, HL7 FHIR из клиник)
- Отслеживание 60+ маркеров: ОАК, биохимия, гормоны, воспаление, метаболизм
- Биологический возраст по крови (алгоритм типа InnerAge 2.0 / Thorne-подход)
- Trend-алёрты: «Твой ферритин падает 3-й тест подряд — вот что это значит»

**Блок микробиома:**
- Партнёрство с лабораториями (импорт результатов shotgun/16S)
- Diversity score, ключевые роды (Akkermansia, Lactobacillus, Faecalibacterium)
- Связка: «Твой генотип FUT2 означает, что тебе нужен Bifidobacterium — посмотрим, есть ли он»
- Диетические рекомендации через призму микробиома

**Блок wearables:**
- Интеграция: Apple Health, Garmin, Oura, WHOOP
- Сон, HRV, VO₂max-прокси, нагрузка

**Алгоритмы:**
- Multi-layer biological age: взвешенная модель из маркеров крови + темп старения (DunedinPACE-вдохновлённый подход)
- Anomaly detection на временных рядах (изоляционный лес / LSTM)
- Корреляционный движок: «Когда ты спишь <6 часов, твой CRP растёт на X%»

---

### Фаза 3 — Personalized Protocol Engine (твой личный протокол)

**Что делает:**
- Генерирует персональный стек рекомендаций — пересечение генетики + текущих маркеров + образа жизни
- Разбивает на 4 домена: питание, движение, нутрицевтики/БАД, восстановление/сон
- Каждая рекомендация объяснена: *«Почему именно тебе»* + *«Что изменится в твоих маркерах»*
- Протоколы ранжированы по impact score и сложности внедрения
- **Ретест-контур:** «Через 3 месяца пересдай это → проверим эффект»

**Алгоритм scoring:**
```
Impact = (genetic_weight × 0.3) + (current_biomarker_delta × 0.4) 
         + (lifestyle_gap × 0.2) + (evidence_strength × 0.1)
```

## AI-слой: Genesis Coach

- **Chat-интерфейс:** задаёшь вопросы своим данным — «Почему у меня падает энергия?», «Что стоит изменить в питании этой осенью?»
- **Narrative reports:** раз в месяц — «Месяц твоего здоровья» — понятный дайджест трендов
- **Алёрты:** не просто «маркер вышел за норму», а контекстный: «Это сочетание маркеров встречается при...»
- **Memory:** помнит твои предыдущие протоколы, что сработало, что нет
