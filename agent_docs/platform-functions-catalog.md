## Легенда колонок

**Статус:**
- ✅ — реализовано и в проде на genesisbio.ru
- 🚧 — в активной разработке или ближайший приоритет
- 📋 — спроектировано (в концепции/архитектуре), не реализовано
- 💡 — идея из переписки/обсуждений, не запроектирована формально

**Контур (по умолчанию, **редактируется при принятии решения по тарифам**):**
- 🛒 — естественно ложится в разовый ген-продукт (открывается один раз, доступ навсегда)
- 🔄 — естественно ложится в подписку (ongoing access, regular updates)
- ↔ — может в любой из двух (вопрос упаковки)
- 🆓 — публичный freemium (без регистрации; SEO-хук)
- ⏸ — Phase 2 по ADR-001 (Семья / Профи / Клиника / мужская ЦА)
