# Пользовательский путь — версия на 2026-05-21

> Снимок состояния продукта на сегодняшний день. Описывает, что реально работает в прототипе, какую ценность получает пользователь и какой результат остаётся в итоге. Обновлять при существенных изменениях функциональности.

## Точки входа (с landing-страницы genesisbio.ru)

С главной страницы (`src/app/page.tsx`) пользователь видит две основные CTA — **«Выбрать тест»** и **«Пройти анкету»**. Через хедер также доступны: **калькулятор биовозраста**, **каталог тестов**, **О нас**, **Контакты**, **Регистрация**.

Фактически реализовано **четыре независимых сценария ценности** + сценарий управления аккаунтом.

## Сценарий 1. Купить генетический тест (E-commerce путь)

**Маршрут:** `/` → `/catalog` → `/catalog/[slug]` → `/cart` → `/checkout` → `/checkout/success`

**Что делает пользователь:**
- Просматривает каталог из 73 тестов, фильтрует по категориям.
- Открывает карточку теста с описанием.
- Добавляет в корзину, оформляет заказ.
- Регистрируется (если не залогинен) для отслеживания статуса.

**Ценность:** возможность реально заказать генетический тест онлайн.

**Результат:**
- Заказ создаётся в БД, виден в `/dashboard/orders/[id]`.
- ⚠️ **Оплата — заглушка** (`api/payment/create` без YooKassa).
- ⚠️ **Email-уведомления не отправляются** (Resend в env, но не подключён).
- Дальнейший процесс (логистика теста, выдача результата) — в офлайне через админку.

## Сценарий 2. Пройти биохакинг-анкету (Soft-gate воронка)

**Маршрут:** `/questionnaire` → `/questionnaire/start` (9 шагов визарда) → `/questionnaire/result/[sessionToken]`

**Что делает пользователь (9 шагов, ~25 мин, ~100 полей):**
1. Intro
2. Личное (пол, возраст, рост/вес, ОТ/ОБ)
3. Образ жизни
4. Анамнез
5. Системы организма
6. Стресс и аллергии
7. Лекарства и БАД
8. Питание
9. Симптомы и цели

Прогресс автосохраняется в LocalStorage — можно прерваться и вернуться без регистрации.

**Ценность:** мгновенно получить персональные подсветки, не платя и не регистрируясь. Это лидогенерационный инструмент.

**Результат на `/questionnaire/result/[sessionToken]`:**
- Панель **подсветок** (15 детерминированных правил из `lib/questionnaire/rules`): дефициты нутриентов, фарм-нагрузка, факторы образа жизни, маркеры стресса.
- Аккордеон с собственными ответами (шаг 1 — человекочитаемо, шаги 2–8 пока в виде сырого JSON).
- Дисклеймер «не медицинский диагноз».
- **CTA на регистрацию** с привязкой `sessionToken` → анкета сохраняется в личный кабинет.

После регистрации: `/dashboard/questionnaire` хранит результат, готовится как контекст для будущего Genesis Coach.

## Сценарий 3. Рассчитать биовозраст (Лид-магнит)

**Маршрут:** `/calculator` (без авторизации)

**Что делает пользователь:** вводит 9 биомаркеров крови (PhenoAge от Levine et al. 2018).

**Результат (`CalculatorResult` компонент):**
- Биологический vs паспортный возраст, разница в годах.
- Цветовая шкала 20–100 с двумя маркерами (паспорт и био).
- Таблица каждого маркера со статусом (good / borderline / bad).
- «Details for nerds» — раскрытие формулы и проблемных показателей.

**Ценность:** SEO-магнит и конкретный полезный инструмент за 2 минуты, без аккаунта. Прямой аналог — платный InsideTracker.

**Результат:** одноразовый расчёт, **сохранения в ЛК нет** (стоит в Приоритете 2 roadmap).

## Сценарий 4. Загрузить ДНК-отчёт CERBALAB (Premium-ценность)

**Маршрут:** регистрация → `/dashboard/dna` → upload PDF → `/dashboard/dna/[id]`

**Что делает пользователь:**
- Загружает PDF-отчёт лаборатории CERBALAB.
- Парсер (`src/lib/dna/cerbalab-parser.ts`) извлекает 223 маркера + метаданные пациента (имя, № образца, дата).

**Результат:**
- Карточка отчёта со счётчиком маркеров, бейджем лаборатории.
- На детальной странице: таблица маркеров, поиск по rs/гену, полнотекстовый поиск по интерпретациям.
- Можно скачать исходный PDF, удалить отчёт.
- ⚠️ Структурированной интерпретации по доменам (метаболизм, детокс, спорт…) **пока нет** — это DNA Parser v2 (Приоритет 2).

## Сценарий 5. Управление аккаунтом

**Маршрут:** `/login` → `/dashboard` → `/dashboard/{orders|profile|dna|questionnaire|results}`

`/dashboard` редиректит на `/orders`. Доступны: история заказов, профиль, ДНК-отчёты, анкета, результаты (заполняются админом вручную через `/admin/results`).

## Что пользователь реально получает сегодня — резюме

| Что обещано | Что есть | Что отсутствует |
|---|---|---|
| Заказать ген-тест | Каталог, корзина, чекаут, заказ в ЛК | Реальная оплата, email-уведомления |
| Понять свой образ жизни | Анкета 9 шагов + 15 авто-подсветок | Человекочитаемые ответы на шагах 2–8 |
| Узнать биовозраст | PhenoAge калькулятор со шкалой | Сохранение, доп. калькуляторы (HOMA-IR, FIB-3 и пр.) |
| Хранить ДНК | Парсинг 223 маркеров CERBALAB, поиск | Интерпретация по 7 доменам, радар, не-CERBALAB форматы |
| Импорт крови, протокол, Genesis Coach | — | Целиком впереди (фазы M4–M12) |

## Главный сюжет, как его проживает пользователь

> «Я зашёл, посчитал свой биовозраст (бесплатно, без регистрации) → понял, что есть проблема → прошёл анкету (бесплатно, увидел подсветки про дефициты и фарм-нагрузку) → захотел углубиться → зарегистрировался → купил ген-тест в каталоге → получил результат от админа → загрузил PDF CERBALAB и вижу свои 223 маркера с поиском.»

## Сильные стороны прототипа

- **Три самостоятельные точки ценности до регистрации** (анкета, калькулятор, каталог) — редкое сочетание для early-stage health-tech.
- Soft-gate анкеты позволяет накапливать ценность анонимно, конверсия в аккаунт происходит после демонстрации результата.
- ДНК-парсер CERBALAB даёт 100% покрытие маркеров — реальное Premium-предложение в ЛК.

## Слабые места, на которые упирается воронка

1. **Оплата-заглушка** обрывает коммерческий цикл — без YooKassa не закрыть первую сделку онлайн.
2. **Нет писем** — после заказа пользователь «выпадает» из контакта.
3. **После загрузки ДНК нет «вау-результата»** по доменам — пока это «база данных маркеров с поиском», а не инсайт.

Эти три блока — самые дешёвые шаги к продукту, который продаёт сам себя.
