## Суть

50 генетических точек — «ДНК-карта пищевых реакций для женщины, которая хочет оставаться стройной, энергичной и ухоженной без жёстких диет». ЦА в формулировке Галины: **женщина 35–60**.

**Якорная боль:**
> «Почему мне уже нельзя питаться "как всем", если я хочу сохранять стройность, энергию, кожу и лёгкость в любом возрасте?»

## Боли для лендинга (формулировки Галины)

- Почему вес стал упрямее — не «плохая сила воли», а индивидуальная реакция на жиры/углеводы/насыщение/омега-баланс.
- Почему «правильное питание» не даёт результата — у каждой свой рычаг (насыщенные жиры / углеводы / омега / кофеин / соль).
- Почему появляются отёки и тяжесть — соль, кофеин, алкоголь, реакция сосудов.
- Почему после 40–45 сложнее восстанавливаться — антиоксидантная защита, глутатион, фолатный цикл, витамин D, железо, ЖК.
- Почему чужие схемы ЗОЖ не подходят — индивидуальная переносимость кофе/молочки/низкоуглеводки.

**Расширение до общих болей (шире женской ЦА, на будущее):** «питаюсь нормально, а энергии нет», «кофе мешает спать», «от молочки вздутие», «худею хуже подруги на той же диете», «холестерин растёт, хотя ем правильно».

## Конкурентное позиционирование против MyGenetics

- **Размер:** 50 ген-точек (база). Сравнение — MyGenetics «Wellness»; их «Эксперт» = 55 генов за **27 900 ₽** (ценовой якорь).
- **Пересечение с MyGenetics** (по «ген + rs-точка», не по названию гена):
  - **19 / 50 (38 %)** — прямое совпадение точек.
  - **31 / 50 (62 %)** — новые/уточнённые точки Сербалаба (по научной литературе).
- **УТП:** «смотрим больше генов, чем Wellness; точки уточнённые → выше валидность, точнее релевантность запроса».
- **Спорт не включён**, но «глубоко про метаболизм».
- **Тактика по чужим отчётам:** клиент загружает свой MyGenetics → совпадение лишь 38 % → «у нас точнее, есть контраргументы» → продаём и опытным, не только новичкам. (Прямая бизнес-логика для функции D4 «загрузка чужих ДНК-отчётов» из каталога.)

## Механика вовлечения: квиз «Генетический чекап питания»

> Идею Галина подчёркивает как свою находку. **Это отдельный лид-генератор**, не наша биохакинг-анкета (анамнез).

**Логика:** не «расскажите проблемы» и не «у вас проблемы — мы закроем», а **«пройди опросник → получи генетический чекап»**. Клиент отмечает **минимум 5 вопросов** → система показывает, какие ген-точки в панели на них отвечают → переход к продаже отчёта (50 точек). Вопросы 16–17 покрывают всю совокупность 50 точек.

## Связь с каталогом функций

- Квиз = **новая функция**, кандидат в раздел C каталога (опросники), отдельная от C1 (биохакинг-анкета). Предлагаемый id: **C18 «Квиз генетический чекап питания»** (freemium лид-генератор → продажа отчёта).
- Базовый ген-блок = генетическое наполнение **Тарифа 1 (или его ядра)**; ложится на E2/E3 (парсинг + группировка по доменам).
- Загрузка чужих MyGenetics (D4) получает прямую бизнес-логику: 38 % совпадение → апсейл «у нас точнее».
