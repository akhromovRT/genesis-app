## Воронка продаж (11 сценариев)

Ключевые маршруты (полная матрица — в источнике):

| Маршрут | Цена ₽ | Консультации |
|---|---|---|
| Полный пакет сразу (1+2+3+4+5) | **65 000** | 2 ч включены |
| База — стоп (блок 1) | 15 900 | нет |
| Сильный старт (1+2 или 1+3, +подарок 4) | 31 800 | нет |
| Wellness-трио (1+2+3 −10%, +подарок 4) | 42 900 | нет |
| Wellness-трио → блок 5 *(ключевая промежуточная)* | 70 200 | 2 ч в подарок |
| Пошаговая полная сборка | 75 000 | 2 ч в подарок |
| Поштучно | 89 900 | нет |

**Ожидаемое распределение:** база 40% / полный пакет 40% / пошаговый апсейл 20%.

**Главный месседж воронки:** чем раньше клиент берёт полный объём, тем выгоднее цена и точнее стратегия биохакинга.

## Подблоки по болям

Полные таблицы клиентских болей по каждому блоку — в `sources/2026-05-22-krasivoe-dolgoletie-tariff.md`. Кратко:
- **Блок 1:** жиры/омега, липиды, углеводы/инсулин, метаболизм/вес, кофеин, соль/отёки, алкоголь, лактоза, железо, витамин D/B, нутриентный чекап, БАД без хаоса, антиоксиданты.
- **Блок 2:** сила/выносливость, тип тренировок, спорт для похудения, восстановление, травмы, суставы, коллаген, костный ресурс, остеопороз, тонус, стратегия движения.
- **Блок 3:** кожа/эстетика возраста, кожный барьер, реакция на косметику, восстановление после процедур, пигментация, воспаление, гликирование, плотность тканей, гормональная эстетика 40+, волосы, безопасность процедур.
- **Блок 4:** сон/циркадные, стресс-реактивность, мотивация/дофамин, нейропластичность.
- **Блок 5:** сердце/сосуды, гемостаз, иммунитет/воспаление, гормонально-репродуктивный профиль, онконастороженность, аптечка (фармакогенетика).
