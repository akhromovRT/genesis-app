# Модуль биохакинг-анкеты — бриф для партнёра

**Дата:** 2026-04-06
**Статус:** Реализовано, задеплоено на genesisbio.ru

---

## 1. Что это

Genesis — платформа персонального здоровья и долголетия. Модуль **биохакинг-анкеты** — это первая точка контакта нового пользователя с системой.

Пользователь проходит структурированный опрос (~25 минут) и получает автоматический анализ: какие дефициты нутриентов вероятны, насколько сбалансирован образ жизни, есть ли красные флаги по лекарствам и стрессу.

**Зачем бизнесу:**
- **Лид-воронка** — анкету можно пройти бесплатно без регистрации. Результат мотивирует зарегистрироваться.
- **Контекст для продаж** — зная ответы, можно предлагать релевантные тесты и услуги.
- **Основа для AI** — в будущем Genesis Coach (AI-ассистент) будет использовать анкету как стартовый контекст для персональных рекомендаций.

**Источник вопросов:** профессиональные анкеты нутрициолога (2 документа: «Образ жизни, Лекарства, БАД» и «Питание и Нутриенты»).

## 4. Что анализируется: 15 правил подсветок

Система анализирует ответы и подсвечивает потенциальные проблемы. Это **не AI, а чёткие правила** — предсказуемые, тестируемые, с обязательными дисклеймерами.

### Дефициты нутриентов (на основе 30 симптомов)

| Подсветка | Когда срабатывает | Рекомендация |
|---|---|---|
| **Дефицит магния** | 3+ из: судороги, тремор, плохой сон, раздражительность | Анализ Mg в эритроцитах; рацион: зелень, орехи, бобовые |
| **Дефицит B12/фолатов** | 3+ из: утомляемость, одышка, бледность, жжение языка, brain fog | Анализ B12 + гомоцистеин; особенно для вегетарианцев |
| **Дефицит железа** | 3+ из: бледность, утомляемость, одышка, выпадение волос, пика | Ферритин, сыв. железо, ОЖСС, гемоглобин |
| **Дефицит витамина D** | 2+ из: боли в костях, слабость мышц, частые простуды | Анализ 25(OH)D; цель 50-80 нг/мл |
| **Дефицит Омега-3** | 2+ симптома сухости + редко жирная рыба | Жирная рыба 2-3 раза/нед или добавка EPA/DHA |
| **Дефицит цинка** | 2+ из: потеря вкуса, медленное заживление, простуды | Цинк в сыворотке; мясо, морепродукты, тыквенные семечки |

### Питание

| Подсветка | Когда срабатывает | Рекомендация |
|---|---|---|
| **Мало клетчатки** | Овощей <250г + редко крупы + упомянуты запоры | 500+ г овощей/день, цельнозерновые, бобовые |
| **Избыток сахара** | Часто сладкое + часто мучное + сильная тяга | Проверить HbA1c, инсулин, HOMA-IR |

### Лекарства

| Подсветка | Когда срабатывает | Рекомендация |
|---|---|---|
| **⚠️ Высокая фарм-нагрузка** | 5+ групп препаратов «часто» | Фармакогенетическое тестирование (CYP450) |
| **Антибиотики + ЖКТ** | Антибиотики регулярно + жалобы ЖКТ | Анализ микробиоты + пробиотики |

### Образ жизни

| Подсветка | Когда срабатывает | Рекомендация |
|---|---|---|
| **Мало воды** | Менее 1 л/день + кофе 3+ чашки | 30 мл на кг массы тела |
| **Сбитые циркадные ритмы** | Еда <3ч до сна + плохой сон | Последний приём за 3ч до сна |
| **Низкая активность** | Уровень = «низкий» | 150-300 мин/нед умеренной + силовые 2 р/нед |
| **⚠️ Высокий стресс** | Уровень 4-5 + плохой сон + раздражительность | HRV-тренинг, дыхательные практики, психотерапевт |
| **Регулярный алкоголь** | 3+ раз в неделю | Проверить АЛТ, АСТ, ГГТ |

**Каждая подсветка содержит:**
- Что именно в ответах послужило триггером (прозрачность)
- Конкретную рекомендацию: какие анализы сдать / что изменить
- Дисклеймер: «Не является медицинским диагнозом. Рекомендуем обсудить со специалистом.»

## 5. Ключевые UX-решения

| Решение | Почему так |
|---|---|
| **Без регистрации** | Не отпугиваем на входе; показываем ценность, потом конвертируем |
| **Автосохранение** | 25 минут — долго, люди прерываются; не потерять прогресс = не потерять лида |
| **Мягкий gate** | Сводку показываем сразу, но «сохранить + подключить анализы» → только через регистрацию |
| **Правила, не AI** | Предсказуемо, дёшево, юридически чисто; AI-интерпретация — в будущих версиях |
| **Обязательные дисклеймеры** | Юридическое требование: не медицинская услуга, не диагноз |

## 6. Что НЕ реализовано (и план)

| Фича | Статус | Когда |
|---|---|---|

| Красивое отображение ответов | Шаги 2-8 в JSON-формате, step1 красиво | Следующая итерация |
| LLM-интерпретация | Только правила, не AI | v2 (Protocol Engine) |
| Повторное прохождение | Только одна версия | По фидбеку |
| Интеграция с анализами крови | Не связано | M4-6 (Biomarker Layer) |

## 7. Дальнейшие шаги

### Ближайшие (1-2 недели)

1. **Тестирование с реальными людьми** — дать 5-10 человекам пройти анкету, собрать фидбек:
   - Понятны ли вопросы?
   - Не слишком ли долго?
   - Ценны ли подсветки?
   - Где «выпадают» пользователи?
2. **Красивая сводка** — заменить JSON на человекочитаемый формат
3. **Email-уведомление** — отправлять сводку на email после завершения

### Среднесрочные (месяц)

4. **Перекрёстные подсветки** — анкета + анализы крови = подтверждённые дефициты (например, симптомы B12 в анкете + низкий B12 в крови = «подтверждённый дефицит B12»)
5. **Валидированные шкалы** — добавить стандартизованные опросники (PHQ-9, PSS, PSQI) как мини-анкеты
6. **Сравнение «было/стало»** — повторное прохождение через 3-6 месяцев

### Стратегические (квартал)

7. **Genesis Coach** — AI-ассистент, который знает анкету: «Вы указали судороги и плохой сон — давайте проверим магний в крови»
8. **Protocol Engine** — полный персональный протокол на основе анкета + генетика + кровь + микробиом
9. **Корпоративный модуль** — массовое анкетирование сотрудников → агрегированный отчёт для HR (без персональных данных)

## 8. Бизнес-метрики для отслеживания

| Метрика | Что покажет |
|---|---|
| Конверсия лендинг → старт визарда | Насколько привлекательно предложение |
| Дропаут по шагам (на каком шаге бросают) | Где анкета «тяжёлая» |
| Процент завершивших из начавших | Общая «проходимость» |
| Конверсия сводка → регистрация | Эффективность soft-gate |
| Среднее кол-во подсветок на пользователя | Полезность правил |
| Какие подсветки срабатывают чаще всего | Приоритеты для контента/продуктов |

*(Google Analytics / Яндекс.Метрика ещё не подключены — в ближайшем плане)*
